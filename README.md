# docker-polemarch

You need to add those lines to you `/etc/hosts` file to resolve the dns names used by this project:

```ini
127.0.0.1  polemarch
127.0.0.1  centrifugo
```

As soon as the compose stack is up and running, visit [http://polemarch:8080](http://polemarch:8080) and login with the default credentials:

```
username: admin
password: admin
```
